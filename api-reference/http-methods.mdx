### `GET`

`GET` methods are used for reading data. Filtering, sorting, or pagination is done via query parameters.

```http
curl "https://api.firmable.com/company?ln_slug=company_slug" \
  -H "Authorization: Bearer <FIRMABLE_API_KEY>"
```
